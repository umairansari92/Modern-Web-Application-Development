JavaScript - Class 01 Summary

What is JavaScript?

JavaScript is a lightweight, interpreted, or JIT-compiled programming language primarily used to create dynamic and interactive content on websites.

What is a Program?

A program is a set of instructions written in a programming language to perform a specific task when executed by a computer.

What is a Programming Language?

A programming language is a formal language comprising a set of instructions that produce various outputs, enabling developers to communicate with computers.

What are Programming Fundamentals?

Programming fundamentals include basic concepts like variables, data types, control structures (loops, conditionals), functions, and error handling that form the foundation of writing code.

Low-Level vs High-Level Programming Languages

  • Low-Level: Closer to machine code, e.g., Assembly, C.
  • High-Level: Abstracted from hardware, e.g., JavaScript, Python.

Difference Between Java and JavaScript

  • Java: Object-oriented, compiled language for server-side applications.
  • JavaScript: Lightweight, interpreted language for client-side web development.

Compiler vs Interpreter

Feature Compiler Interpreter
Execution Compiles the entire program before execution. Translates and executes code line by line.
Speed Faster after compilation. Slower due to line-by-line execution.
Error Handling Displays all errors after compilation. Stops at the first error encountered.

JavaScript uses Just-In-Time (JIT) Compilation, blending interpretation and compilation for efficiency.

JavaScript Key Features

  • Lightweight and interpreted.
  • Highly flexible and dynamic.
  • Prototype-based object orientation.
  • Event-driven and asynchronous programming support.

JavaScript Frameworks

Popular frameworks include React, Angular, Vue.js, and Node.js, each offering unique tools for web development.

JavaScript Syntax

JavaScript syntax refers to the rules and structure for writing JavaScript code, including keywords, operators, and punctuation.

Variables in JavaScript

Variables store data values. Declared using var, let, or const. For example, let x = 10;.
